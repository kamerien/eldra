# MTG VRChat ワールド ユーザーガイド

## 1. はじめに

### 1.1 システム要件

```yaml
VR:
    推奨:
        - Oculus Quest 2/3
        - Valve Index
        - HTC Vive
    最小:
        - Oculus Rift S
        - Windows Mixed Reality

デスクトップ:
    推奨:
        CPU: i5-9400以上
        GPU: GTX 1660以上
        RAM: 16GB以上
    最小:
        CPU: i5-6500
        GPU: GTX 1060 3GB
        RAM: 8GB
```

### 1.2 基本操作方法

```yaml
VRモード:
    カード操作:
        - グラブ: トリガー
        - 回転: グリップ + ジョイスティック
        - めくる: トリガー + グリップ

    メニュー操作:
        - 表示: 左手メニューボタン
        - 選択: トリガー
        - スクロール: ジョイスティック

デスクトップモード:
    カード操作:
        - 掴む: 左クリック
        - 回転: 右クリック + ドラッグ
        - めくる: 右クリック

    メニュー操作:
        - 表示: Tab
        - 選択: 左クリック
        - スクロール: マウスホイール
```

## 2. ゲームの開始

### 2.1 デッキ構築

```yaml
1. デッキビルダーの起動:
    VR: 左手メニュー → [Deck Builder]
    PC: Tab → [Deck Builder]

2. カード検索:
    - フォーマット選択
    - 検索条件入力
    - フィルター適用

3. デッキ編集:
    - カードをドラッグ＆ドロップ
    - 枚数調整（+/-ボタン）
    - デッキリスト確認

4. デッキの保存:
    - デッキ名入力
    - フォーマット確認
    - [Save]ボタンクリック
```

### 2.2 対戦開始

```yaml
1. テーブルの選択:
    - 空きテーブルを探す
    - 着席ボタンを押す
    - デッキを選択

2. 準備:
    - 初期手札の確認
    - マリガン判断
    - ライフポイント設定

3. ゲーム開始:
    - 先攻/後攻決定
    - 手札の確定
    - ゲーム開始ボタン
```

## 3. ゲームプレイ

### 3.1 基本アクション

```yaml
カード操作:
    ドロー:
        VR: デッキに手を伸ばしてトリガー
        PC: デッキを左クリック

    プレイ:
        VR: カードを場に置く
        PC: カードをドラッグ＆ドロップ

    タップ:
        VR: カードをグラブして90度回転
        PC: カードを右クリック

ライフポイント:
    変更:
        VR: 数字パネルをポイント＆クリック
        PC: +/-ボタンをクリック
```

### 3.2 特殊アクション

```yaml
スタック操作:
    呪文を唱える: 1. カードを手札から場に出す
        2. コストを支払う
        3. 効果を選択

    効果を積む: 1. アクティブなカードを選択
        2. 効果を選択
        3. 対象を指定

インスタント:
    優先権の処理:
        - パス: 緑ボタン
        - ホールド: 赤ボタン
        - アクション: カードをプレイ
```

## 4. 高度な機能

```yaml
観戦モード:
    入り方: 1. テーブル近くの観戦エリアに移動

リプレイ機能:

        - ハイライト登録

    再生:
        - タイムライン操作
        - 速度調整
        - 視点切替
```

```yaml
カウンター管理:

トークン作成:

        - サイズ調整
        - 効果付与
```

## 5. トラブルシューティング

### 5.1 一般的な問題

```yaml
表示の問題:
    カードが表示されない: 1. メニューを開き直す
        2. テーブルをリセット
        3. ワールドを再読み込み

    テクスチャがぼやける: 1. グラフィック設定を確認
        2. キャッシュをクリア
        3. VRChatを再起動

操作の問題:
    カードを掴めない: 1. 位置を調整
        2. ピックアップ距離を確認
        3. コントローラーの再キャリブレーション

    メニューが反応しない: 1. UIをリフレッシュ
        2. 別の場所に移動
        3. インスタンスを再参加
```

### 5.2 ネットワーク問題

```yaml
同期の問題:
    カードの位置ずれ: 1. カードをリセット
        2. テーブルをリセット
        3. インスタンスを再参加

    操作の遅延: 1. ネットワーク状態を確認
        2. 他のアプリを終了
        3. VRChatを再起動

切断:
    再接続手順: 1. ゲーム状態を保存
        2. インスタンスに再参加
        3. デッキを再ロード
```
